
------------------------------------------------------------------------------------
		CopyMoveFile CmdLine Utility v[VERSION]
------------------------------------------------------------------------------------

Allows copying or moving files from the command line.

USAGE:  CopyMoveFile <action> /inPath <path> [/outPath <path>]

  action	Either 'copy' or 'move'.
  inPath	Path of the source file to copy/move.
  outPath	Path of the destination directory or file name (optionally
			renaming the file). If not specified, a file selection dialog will be
			displayed.

All paths are assumed to be in the current directory if a full path isn't specified.
------------------------------------------------------------------------------------
